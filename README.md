# Fish Species Classification using Machine Learning

## Project Overview
This project focuses on classifying fish species using machine learning. It covers preprocessing, exploratory data analysis, model training, evaluation, and prediction.

## Objective
- Classify fish species.
- Analyze and preprocess data.
- Train and evaluate machine learning models.

## Dataset
**Features:**
- Weight
- Length1
- Length2
- Length3
- Height
- Width

**Target:**
- Species

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Workflow
1. Import libraries.
2. Load the dataset.
3. Preprocess the data.
4. Perform Exploratory Data Analysis (EDA).
5. Train the classification model.
6. Evaluate the model using Accuracy, Confusion Matrix, and Classification Report.
7. Predict fish species.

## Conclusion
This project demonstrates an end-to-end machine learning classification pipeline for identifying different fish species based on their physical characteristics such as weight, length, height, and width. The project includes data preprocessing, exploratory data analysis, model training, evaluation, and prediction, providing practical experience in applying machine learning techniques to a real-world classification problem.
